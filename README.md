# Suyoung Kwon's Portfolio

This is my personal design portfolio website.

## Tech Stack
- **Static Site Generator**: [Jekyll](https://jekyllrb.com/)
- **Theme**: [al-folio](https://github.com/alshedivat/al-folio)
- **Deployment**: GitHub Pages via GitHub Actions

## Structure
- `_pages/`: Main pages (Home, Projects, CV, etc.)
- `_projects/`: Project details in Markdown
- `assets/img/projects/`: Project images
- `_config.yml`: Site configuration
